# Scripts

This directory contains the core analysis scripts for the study.

## Contents

| Script | Purpose |
|--------|---------|
| `pipeline_lib.py` | Shared utilities and constants |
| `build_prisma_flow.py` | PRISMA 2020 flow diagram |
| `build_evidence_map.py` | Evidence map visualisation |
| `clinical_readiness.py` | Scenario-level clinical readiness assessment |
| `statistical_analysis.py` | Temporal trends and descriptive statistics |
| `meta_analysis.py` | Study-level performance heterogeneity |
| `prepare_meta_input.py` | Prepares data for meta-analysis |
| `tripod_llm_scoring.py` | TRIPOD-LLM item-level scoring |
| `rob_assessment.py` | Formal methodological assessment (QUADAS-3 / PROBAST+AI) |
| `publication_form_audit.py` | Publication-form classification audit |
| `compare_protocols.py` | Primary vs. supplementary stream comparison |
| `generate_manuscript_figures_html_svg.py` | SVG/HTML figure generation |

Internal submission build scripts, QC wrappers, and pipeline orchestration tools are not included.
